# Code of Conduct

## Our Standard

This project operates under a simple standard: **professionalism, respect, and quality**.

Inspired by the Night Stalker Creed: success as standard, precision in execution, quiet professionalism in conduct.

## Expectations

- Be constructive in feedback
- Respect the codebase conventions (DryRun-first, KB units, deprecate-never-delete)
- Test before submitting
- Document your changes

## Enforcement

This is a small project maintained by one person. Issues will be handled directly and fairly.
